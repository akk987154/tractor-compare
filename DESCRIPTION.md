# TractorCompare

TractorCompare 是一个拖拉机规格对比工具，支持不同品牌和型号之间的并排比较，并通过雷达图展示关键参数差异。

## 核心内容

- 支持 John Deere、Kubota、Massey Ferguson、Case IH、New Holland 等品牌
- 提供拖拉机参数并排对比
- 使用雷达图展示性能差异
- 支持筛选、搜索和对比页浏览

## 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- Recharts

## 运行方式

```bash
npm install
npm run dev
```
